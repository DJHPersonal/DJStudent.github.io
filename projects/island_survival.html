Island Survival

Unreal Engine 4 C++ Multiplayer Procedural Terrain Group of 2
View code on Github Download and play from itch.io

Things I Implemented

  • Entire procedural terrain system, including biomes and object spawning
  • Majority of LAN multiplayer system
  • Fuel pickup system
  • Main menu and lobby UI elements

Created as a second-year project, my goal with the terrain generation was to develop a highly customizable and reusable system that would speed up the development of future projects. This required thorough foresight to ensure the creation of a modular system which would be easy to use within the editor.

Along with this I was a main developer on the multiplayer system which required many optimization stages to ensure lag, largely due to unnecessary replications of the AI, could be reduced. Due to the terrains high network cost, it was decided that only the vital information, such as its seed, would be sent to the clientâ€™s machines so that they could all independently generate the same terrain.
